AngerScoot

AngerScoot est un projet réalisé en binôme dans le cadre de ma première année de BTS SIO. L’objectif principal était de concevoir un système complet de gestion pour un réseau de bornes de recharge de trottinettes électriques et de scooter à Angers.

Le projet était divisé en deux parties complémentaires :

  • Un site web développé en HTML, CSS, JavaScript et PHP côté serveur, permettant de gérer dynamiquement les données via un back-end connecté à une base de données.
  • Une application de bureau développée en C# (Windows Forms), jouant le rôle d'application client pour accéder et interagir avec les données.

La base de données MySQL était hébergée sur une machine virtuelle Debian, configurée comme un serveur accessible en réseau local via le Wi-Fi du BTS SIO. Cela permettait aux élèves de se connecter au site web hébergé sur ce serveur depuis n'importe quel poste du campus.

Ce projet m’a permis de mettre en place une architecture complète avec un back-end fonctionnel en PHP, d’utiliser une base de données relationnelle, et de collaborer efficacement à deux en répartissant les rôles entre développement web et développement applicatif.
